<div align="center".
	<h1>ytmous</h1>
	A anonymous Youtube that does not spy on you.
</div>

## Feature
- 0% Ads
- 0% Tracker
- Download Youtube Videos while you watch them

## Installation
```bash
git clone https://github.com/Yonle/ytmous
cd ytmous
npm install
```

## Starting Server
```bash
npm start
```
Or
```bash
PORT=8080 npm start
```
