## Installation
```bash
git clone https://github.com/Yonle/ytmous
cd ytmous
npm install
```

## Starting Server
```bash
npm start
```
Or
```bash
PORT=8080 npm start
```

## NOTE
In Browser based **Chromium**, Seeking Video may not work properly. It's recommended to use **Firefox** for Streaming Video.
